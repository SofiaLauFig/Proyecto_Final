Resumen HTML + CSS

1. Estructura básica de un archivo HTML

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Mi página</title>
  </head>
  <body>
    <!-- Contenido visible -->
  </body>
</html>

2. CSS embebido (dentro del mismo HTML)

<style>
  body {
    background-color: lightblue;
    font-family: Arial, sans-serif;
  }
</style>

3. CSS externo (archivo separado)

  • Crear un archivo llamado styles.css
  • Escribir el CSS dentro de ese archivo:
body {
  background-color: #f0f8ff;
  font-family: sans-serif;
}
  • Luego, enlazarlo desde el HTML con:
<link rel="stylesheet" href="styles.css">

4. Orden de lectura del navegador

El navegador lee primero el CSS externo y luego el embebido. Si hay estilos repetidos, el último que aparece gana.

5. ¿Cuándo usar cada uno?

  • ✅ CSS embebido: ideal para prácticas y tareas cortas
  • ✅ CSS externo: ideal para proyectos largos o páginas múltiples
  • ⚠️ Podés usar ambos, pero el orden importa
  • ⚠️ CSS en línea (style="...") se usa solo para casos puntuales, no recomendable en proyectos grandes
